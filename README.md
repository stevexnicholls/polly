#### Prerequisites

* [Git](https://git-scm.com/)
* [Go](https://golang.org/dl/)

#### Fetch from GitHub

```bash
mkdir $HOME/src
cd $HOME/src
git clone https://github.com/stevexnicholls/polly.git
cd hugo
go install
```

**If you are a Windows user, substitute the `$HOME` environment variable above with `%USERPROFILE%`.**
